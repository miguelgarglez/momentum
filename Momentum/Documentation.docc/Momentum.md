# ``Momentum``

Momentum es una app macOS enfocada en transformar tiempo en progreso. Esta documentación DocC resume las piezas principales del módulo y enlaza a guías de arquitectura, diagnósticos y pruebas.

## Configuración

La pantalla de ajustes usa un layout dividido con una barra lateral de secciones y un panel de detalle para agrupar las preferencias principales (tracking, privacidad, apariencia y más).
